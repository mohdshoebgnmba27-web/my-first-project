# Restaurant QR Ordering SaaS

A modern, full-stack SaaS platform for restaurant QR-based table ordering with real-time order tracking and management.

## 🚀 Quick Start

This project is being built with:
- **Frontend:** Next.js 14 + React + TypeScript + Tailwind CSS
- **Backend:** Node.js + Express.js
- **Database:** PostgreSQL + Prisma ORM

## 📋 Portals

1. **Customer Portal** - QR-based menu browsing and ordering
2. **Owner Dashboard** - Restaurant management and order tracking
3. **Admin Panel** - Platform management and analytics
